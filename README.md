# Zegen Solusi Mandiri Pre-Test Project

## Overview
Proyek ini adalah aplikasi mobile yang dikembangkan menggunakan framework **React Native**. Proses coding dilakukan melalui **Visual Studio Code** dan **Android Studio** yang digunakan sebagai environment untuk menjalankan simulasi dan integrasi SDK

## Technical Stack
* **Framework:** React Native
* **IDE (Coding):** Visual Studio Code (VSC)
* **IDE (Simulation):** Android Studio 2023.3.1.19
* **Platform:** Android
* **SDK Version:** API 35
* **Language:** Java (Android Native side) / JavaScript (React Native)

## How to Run (Simulated)
1. Buka project di Visual Studio Code
2. Jalankan Metro Bundler
3. Buka folder `android` melalui Android Studio untuk melakukan sinkronisasi Gradle dan menjalankan emulator/device fisik dengan API 35
